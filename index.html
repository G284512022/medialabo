Web コンテンツ作成

本テーマの学習の目的

本テーマの学習目的は7回の授業でHTML、CSS、JavaScriptを学び、これらを組み合わせて Web ページを構成する方法を学びます、最後に、学んだことを活用し、自分でWWW のサーバへ 検索し，サーバから届いたデータを Webページとするような Web プロジェクト課題を作成する。

プロジェクト課題について

プロジェクト課題の内容はWWW のサーバへ検索し，サーバから届いたデータを Web ページとして整形するような Web アプリケーションを作成する、僕は世界の天気のテーマを選びました、第一回で学んだWWWとHTMLでプロジェクト課題のタイトル、 文字、を表示する HTMLを作りました、第二回で学んだCSSで課題の画面をデザイン装飾、第三回から第五回で学んだJavaScript で検索欄または検索ボタンを設置しました。最後、第六回で学んだAJAX通信を利用し、サーバから届いたデータを使って課題を完成しました。

プロジェクト

数当でゲーム

数当てゲームは第三回から第五回まで学んだ内容を作ったゲームです、このゲームの仕様は乱数で1から10の値を一つ決めて答えとする、 ユーザは答えを予想して,数を最大で3回入力できる、ユーザの入力が外れた場合は,入力よりも答えが大きいか小さいか表示する3回以内 に答えの数を当てればユーザの勝ち,そうでなければ負け。第三回は学びましたJavaScriptの基礎でMath.random()関数で答えの数値 を指定する、変数yosoは常に4にして、変数と答えの結果を判定して結果をコンソールプログラム作成しました。第四回はDOMを勉強し、 結果をコンソールではなく、ページに表示するためのメソッドhantei()を作りました、最後第五回勉強したイベント処理でラベル， テキストボックス，回答ボタンを設置してゲームを完成しました。

本テーマの学習の感想

今回のWebコンテンツ授業でいろんなウェブサイトの作成に関する知識を学びました、第一回の授業のテーマはWWWとHTML、WWWの部分でURLはプロトコル、サーバ、パスで構成、WEBページはHTML、CSS、JavaScriptと画像・音声・動画で構成したなどWWWに関してとても基礎な知識を初めて知りました。続き今までJavaみたいなプログラミング言語ではなく、HTMLと言うマークアップ言語をはじめて勉強しました、ここで要素属性との書き方、HTMLの構成、HTML のデバッグを学んだ後にこの「Web コンテンツ作成」の活動を記録するindex.htmlの書式と文字の内容を作成しました、第二回のテーマCSSは前回勉強したHTMLの要素それぞれの見た目の設定言い換える装飾を学びます、CSS を HTML に適用した後に、HTMLテキストの色、大きさ、フォント、リストの箇条書きの記号や数値などことを設定できる、最後は今回で学んだCSS知識を使って前回作ったIndex.htmlをちゃんと装飾しました。第三回の授業内容はJavaScript、わたしは今までプログラミング言語Javaとpython学んだことあります、JavaScriptの名前を聞いたことありますが、本格の勉強は今回初めです、JavaScriptの名前はJavaと似ていますが、全く違うプログラミング言語でした、今回の授業で基本的な JavaScript の構文と Java の構文の違いを学び、基礎なプログラミングまたはコンソールのやり方を勉強しました。第四回の授業はDOM プログラミングです、DOM は Web コンテンツを表す木構造、DOMを使うと新しい要素の追加、Webページテキストデータの読み込みなどできます。第五回の授業はイベント処理、イベントとはWeb ページ上で起こった動作や出来事のこと、DOM とイベント処理を組み合わせるとページ上のボタンを押すと、ページの表示内容が変化する、サーバとの通信が完了したときに、受信したデータをページに表示するなどの機能できますなど重要なことできます、例えばinput要素とボタンを使って入力フォームのページができます。第三回から第五回の知識を使って、テキストボックスとボタンが設置されている判定できる数当てゲームを作りました。授業の最後、第六回で通信処理を学び、第一回から第六回まですべて勉強した知識を使うプロジェクト課題を完成し、とても充実でしだ。
